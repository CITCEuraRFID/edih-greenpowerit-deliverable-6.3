# LightFM Recommender Component

## Purpose
Implements a reusable training pipeline for LightFM matrix‑factorization models. The component loads preprocessed data, trains a recommendation model, and persists it for later inference. It is designed to be easily integrated into larger recommendation pipelines.

## Technologies
Python, LightFM, pandas, pickle

## Installation

Create a virtual environment in the project folder

```bash
python -m venv .venv

# Activate the environment
# macOS / Linux
source .venv/bin/activate
# Windows (cmd)
.venv\Scripts\activate.bat
# Windows (PowerShell)
.venv\Scripts\Activate.ps1
```
Install required libraries
```bash
# (Optional) Upgrade pip inside the venv
python -m pip install --upgrade pip

pip install -r requirements.txt
```

## Usage

Training:
```bash
python train.py path/to/processed.csv
```
> Produces lightfm_model.pkl in the working directory

Inference:
```bash
python predict.py --data processed.csv --model lightfm_model.pkl --user <user_id> [--top_n N]
```

### Features
  - Load CSV data using pandas
  - Fill missing values with zeros
  - Simple extension point for more complex preprocessing steps

### Key Functions
  - `load_processed_data`: Reads the CSV generated by the preprocessing component.
  - `train_model`: Creates a LightFM dataset, trains the model, and returns the trained object.
  - Model persistence using pickle.
